1. **Clone the repository**
   ```bash
   git clone https://github.com/pankajkoree/portfoliopk.git
   cd portfoliopk
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or

   # or

   ```

   ```bash

   # or

   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result
